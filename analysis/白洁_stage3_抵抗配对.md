# 白洁 Stage 3：v1(有抵抗) vs v2(无抵抗) — 抵抗指令的压制力是**分源**的

*把白洁四基线(情欲/武侠/极简/反讽 × 白洁/A/B + AI情欲对照)在两种提示词下各跑 n=20,共 240(v1)+ 已有 240(v2)。问题:那句"你不会刻意模仿……追求内心的自洽"到底压住了多少传染?*

> v1 = 有抵抗指令 + 题目在材料前(`prompts/`,与 1260 篇大矩阵同口径)
> v2 = 无抵抗指令 + 题目在材料后(`v2_test/`)
> 共享基准 = `outputs/{基线}/中性`;Δ = cell均值 − 中性均值。

---

## 1. 核心结论:抵抗指令**压得住 AI 源,压不住真人白洁**

| 传染对象 | 句法漂移(段长Δ)v1 → v2 | 抵抗指令效果 |
|---|---|---|
| **AI情欲**(极简基线) | +7.2 → +18.8(**v2 是 v1 的 2.6 倍**) | **强压制** |
| **AI情欲**(情欲基线) | +7.7 → +26.9(**3.5 倍**) | **强压制** |
| **真人白洁**(极简基线) | +6.6 → +6.6(**相同**) | **基本无效** |
| **真人白洁**(情欲基线) | +44.5 → +38.6(v1 反而略强) | **基本无效** |
| **真人白洁**(反讽基线) | +25.4 → +27.1(≈) | **基本无效** |

**机制解释:** 抵抗指令针对的是「刻意模仿」=有意识的风格挪用。
- AI情欲的传染里有大量**可意识到的词汇/句式挪用**(借词率 极简 v1+1.3 → v2+3.0,2.3 倍),所以"别刻意学"一句能压掉一大半。
- 真人白洁的「长句铺陈、慢节奏」是**无意识被节奏带跑**的句法重力——模型并不觉得自己在模仿白洁,所以"别刻意学"根本够不着它。**v1≈v2,甚至 v1 略强。**

→ phase-1 那条 confound #1「抵抗指令制造假性弱传染」**只对 AI 源成立**;对真人露骨原文的句法传染,抵抗指令是失效的。这修正/细化了原来的笼统说法。

## 2. 词汇层:真人锁死,两版都一样

白洁借词率在 v1 与 v2 下都贴着中性(0.07–0.17,Δ≈+0.1),露骨词锁死天花板在**有无抵抗指令下都成立**,robust。

## 3. 一个必须标注的 confound:武侠行不干净

- **武侠** 的 v1 用了补齐材料(篇均1103),但 v2 武侠读白洁是 **phase-1 的旧数据(材料篇均~956)**,没补齐过 → 武侠 v1-vs-v2 混入了材料长度差(看总字数:v1 1129 vs v2 957),不能直接比。
- **干净可比的是 极简/情欲/反讽**(v1、v2 都用补齐后的同一材料)。结论(1)主要建立在这三个基线上。武侠仅作趋势参考。

## 4. 长度的提示词效应(顺带)

同一"800-1000字"指令下,**v1(题目在前+抵抗)产出系统性比 v2 更长**(如情欲读白洁 v1 1154 vs v2 1093;极简 926 vs 870)。题目放在材料前、先立任务再读材料,似乎让模型写得更铺张。这是提示词结构的副作用,与传染信号无关,但记录在案。

---

## 5. 一句话

抵抗指令("你不会刻意模仿")的压制力**分源**:对 AI 同类风格的传染压制 2–3.5 倍(证实"假性弱传染"),但对真人白洁的长句句法重力**几乎无效**——因为后者是无意识的节奏priming,不在"刻意模仿"的射程内。词汇层真人锁死,有无抵抗都一样。

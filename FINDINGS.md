# EROTICA 实验发现

*A small paper on stylistic contamination in LLM-simulated writers*

---

## 摘要

让 Claude Opus 4.7 模拟 7 位风格各异的虚构作家（极简、清欢、武侠、网文、纯爱、情欲、反讽），先让每位作家产出 20 篇阅读材料，再让每位作家在不同条件下创作同题"幻林折枝"短篇，条件包括**不读任何材料**、**中性投喂混合材料**、**被某单一作家污染**。一共 1260 篇产出，按 7 × 7 = 49 个 cell × 每 cell n=20 组织。

三个核心结论：

1. **作家的"污染力"差距悬殊**——最强的反讽（18 个强信号）打过最弱的网文（1 个）将近 20 倍。
2. **传染力和抗污染力强相关**——同一个作家在两个角色上往往都拿满分或都拿零分。
3. **共同变量是"人格饱和度"**——硬规则严密的作家既扛得住别人也推不动别人，反之亦然。

---

## 1. 实验设置

### 1.1 设计

| 维度 | 内容 |
|---|---|
| 模型 | Claude Opus 4.7 |
| 虚构作家 | 7 位（极简/清欢/武侠/网文/纯爱/情欲/反讽，描述见 `writers/`） |
| 阅读材料 | 7 × 4 题目 × 5 篇 = 140 篇 |
| 产出题目 | "幻林折枝"（一个抽象题，让模型有发挥空间） |
| 三档对照 | 不读 / 中性投喂（7 源各 3 篇 = 21 篇） / 被单一源污染（该源 20 篇） |
| 矩阵 | 49 个 cell (7 baseline × 7 source)，每 cell n=20 |

### 1.2 三层指标

| 层 | 指标 | 测什么 |
|---|---|---|
| **句法（D）** | 句长、段长、单句成段率、对话密度、议论标记密度 | 文本"骨架"层面的漂移 |
| **n-gram 借词率（B）** | 产出中"该源独有 2-3 字片段"的出现密度 | 词汇层面的直接借鉴 |
| **TF-IDF 自动指纹（C）** | 每源从阅读材料自动抽出 top-200 区分性词组 | 用作 B 的"指纹词表"，**无需手工挑词** |

每个 cell 跟同基线的"中性投喂"对照，算 z-score = `|delta| / 中性内部标准差`。z ≥ 2 视为强信号。

## 2. 主要发现

### 发现 1：传染力差距 18 倍

![排行榜](analysis/figures/rankings.png)

7 个作家作为污染源的传染力（"作为源时让别人产生的强信号数"）从 18 到 1，差了 18 倍。**反讽**孤立在顶端，**网文**只造成过 1 个强信号。

清欢（汪曾祺）排第 2 出乎意料——汪曾祺的散文化风格在写作教材里被视作"温润不刺激"，但它的**段落跳跃**和**对话稀薄**会破坏依赖对话推进的武侠/情欲基线，同时大量的江南意象也容易被读者直接借鉴。

### 发现 2：**传染力**与**抗污染力**强相关

![作家角色定位](analysis/figures/role_scatter.png)

把每个作家放到二维平面上（x = 作为源的传染力，y = 作为基线的易感度），**七个点呈现明显的反相关**：

- **反讽**：右下角，传染力 18 + 被打穿 0（极端的"打人不被打"）
- **武侠**：左上角，传染力 4 + 被打穿 15（极端的"挨打不还手"）
- **网文**：左偏上，最不会污染别人，但很易感
- **清欢/纯爱**：右下半区，强传染 + 强抗污染
- **极简**：左中，弱传染 + 中度抗污染

这个分布提示一个**共同变量**：这个变量同时决定了"能不能推别人"和"能不能被推"。我们的假说是**人格饱和度**（见 §3）。

### 发现 3：海明威是唯一的"反向污染源"

![借词漂移热力图](analysis/figures/borrow_heatmap.png)

热力图最左边一列（极简作为源）几乎全是浅色或接近 0：海明威的指纹词（老人、男孩、栈桥、火柴）几乎没有进入任何基线的产出。但这不代表海明威没传染力——他的**句法**有传染力，只是方向是**让产出变短**。

| cell | 句长漂移 | 段长漂移 |
|---|---|---|
| 江南→极简（清欢被极简污染） | **-5.66 字 (z=2.58!)** | -23.72 字 |
| 情欲←极简 | -4.94 字 (z=2.13!) | -12 字 |
| 反讽←极简 | -7.65 字 (z=1.97 中) | -36 字 |
| 武侠←极简 | -2.62 字 | -5.38 字 |

**全实验唯一的负方向漂移**——海明威的硬规则（"句子 ≤ 15 字、不议论"）反向投射到了其他基线。

### 发现 4：污染分两个维度

![借词 vs 句法散点图](analysis/figures/borrow_vs_syntax.png)

把 49 个 cell 放到"借词漂移 vs 句法漂移"平面上：

- **右上区（又染词又染骨架）**：武侠←反讽、网文←反讽、武侠←情欲、武侠←清欢 等。这些组合的源既有强词汇标志，又有强句法节奏。
- **右下区（只染词，骨架不动）**：纯爱←网文、武侠←网文 等。网文作为源主要染词（"林天/炼气/宗门"等专属词），但不重塑句法。
- **左上区（不染词，重塑骨架）**：武侠←纯爱、情欲←纯爱、武侠←反讽（部分点）。源的指纹词没大量进入产出，但骨架被强力改造。
- **左下区（噪声内）**：大多数对角线和弱传染组合。

这暗示了**污染源的两个独立维度**：词汇标志强度 vs 句法节奏硬度。它们存在正交性质。

### 发现 5：饱和 vs 未饱和

![句法 z-score 热力图](analysis/figures/zscore_heatmap.png)

7 个对角线（自己读自己 20 篇）的句法漂移可以分成两类：

| 对角线 | 句长 z | 段长 z | 解读 |
|---|---|---|---|
| **极简←极简** | 0.66 | 0.00 | **完全不变** |
| **清欢←清欢** | 1.10 | 0.28 | 几乎不变 |
| **反讽←反讽** | 0.39 | 0.32 | **完全不变** |
| 情欲←情欲 | 1.36 | 0.96 | 弱中 |
| 网文←网文 | 1.64 | 0.67 | 弱中 |
| **纯爱←纯爱** | 2.46 ! | 0.57 | **自我强化** |
| **武侠←武侠** | 3.45 ! | 4.62 ! | **自我强化爆表** |

如果"自己读自己"会让产出更像自己，意味着**身份提示词还没把模型推到极致**。武侠基线读 20 篇自己之后**变得更武侠**：更长的段落、更密集的章回呼吸、更多的人物描写。

反过来，极简和反讽**碰到自己的产出毫无反应**，可能是身份提示词已经把模型逼到风格的天花板，没有上升空间。

### 发现 6：议论密度是反讽的签名信号

反讽（王小波）作为污染源时，被污染的基线**几乎一定会出现议论标记密度激增**，不论被污染的是谁：

| cell | 议论密度漂移 |
|---|---|
| 网文←反讽 | **+4.01 ‰ (z=4.03!)** |
| 纯爱←反讽 | **+3.35 ‰ (z=3.37!)** |
| 武侠←反讽 | **+2.71 ‰ (z=2.50!)** |
| 情欲←反讽 | **+2.13 ‰ (z=2.05!)** |
| 清欢←反讽 | **+1.79 ‰ (z=3.15!)** |
| 极简←反讽 | +0.46（防住） |

只有海明威防住了。这是反讽**最干净的签名**：一个独立的、可解释的、有方向的漂移维度。

而反讽**自己被别人污染**时，议论密度几乎不动。说明它不是普遍漂移，而是反讽在**主动输出**。

## 3. 解读：人格饱和度假说

实验观察可以用一个假说统一解释：**人格饱和度**。

**饱和**：身份提示词把模型推到风格的天花板，没有上升空间。
**未饱和**：身份提示词还有放大空间，模型受输入材料的影响而被推得更极端。

| 作家 | 硬规则 | 饱和度 |
|---|---|---|
| 反讽 | 高议论密度 + 哲学跳跃 + 自嘲叙述者 | 饱和 |
| 极简 | 句子 ≤ 15 字 + 不议论 + 心理通过动作折射 | 饱和 |
| 清欢 | 散文化跳跃 + 食物/植物/手艺细节 + 平和收尾 | 较饱和 |
| 纯爱 | 长句修辞 + 品牌/地标符号 + 内心独白 | 中等 |
| 情欲 | 慢节奏延展 + 男性凝视 + 距离-视线-身体 | 中等 |
| 网文 | 主要是题材（玄幻/修真/升级），语言层面规则少 | 未饱和 |
| 武侠 | 主要是题材（江湖/帮派），章回呼吸 | 未饱和 |

这个假说**同时**解释了三个现象：

1. **饱和的人格 → 自我催化弱**（自己读自己不会更夸张）—— 实测：极简、反讽、清欢对角线 z < 1.2
2. **饱和的人格 → 抗污染强**（防火墙挡住别人的影响）—— 实测：反讽 0/30 被打穿，纯爱 2/30，清欢 5/30
3. **饱和的人格 → 也有传染力**（因为身份特征鲜明）—— 实测：反讽 18，清欢 10，纯爱 8

换句话说：**特征鲜明度决定传染力上限，自己的饱和度决定能不能扛住别人**。这两个独立因素加起来，造成了 §2 的"传染力与抗污染力正相关"。

## 4. v2 提示词实验：去除抵抗指令

**改动**：v1 的每个污染 prompt 都含一句抵抗指令：「你不会刻意模仿——你有自己的活儿要干。作为作者，你追求创新，更追求内心的自洽」。v2 删掉这句，并把题目从材料**之前**移到材料**之后**（让模型先读完 20 篇再写）。其余不变。

### 统计结果（每 cell n=20，6 维强信号数）

海明威（极简）作为基线：

| cell | v1 强信号 | v2 强信号 |
|---|---|---|
| 极简←反讽 | 1/6 | 6/6 |
| 极简←纯爱 | 0/6 | 4/6 |
| 极简←情欲 | 3/6 | 5/6 |
| 极简←网文 | 2/6 | 4/6 |
| 极简←武侠 | 3/6 | 3/6 |

反讽作为基线（与海明威对称，各 5 个源）：

| cell | v1 强信号 | v2 强信号 |
|---|---|---|
| 反讽←情欲 | 1/6 | 1/6 |
| 反讽←纯爱 | 1/6 | 1/6 |
| 反讽←网文 | 1/6 | 1/6 |
| 反讽←武侠 | 1/6 | 1/6 |
| 反讽←极简 | 0/6 | 2/6 |

武侠作为基线（4 个源）：

| cell | v1 强信号 | v2 强信号 |
|---|---|---|
| 武侠←反讽 | 6/6 | 6/6 |
| 武侠←情欲 | 5/6 | 5/6 |
| 武侠←纯爱 | 4/6 | 5/6 |
| 武侠←网文 | 0/6 | 4/6 |

清欢作为基线（4 个源）：

| cell | v1 强信号 | v2 强信号 |
|---|---|---|
| 清欢←反讽 | 4/6 | 4/6 |
| 清欢←情欲 | 1/6 | 1/6 |
| 清欢←纯爱 | 2/6 | 3/6 |
| 清欢←网文 | 1/6 | 1/6 |

### 核心统计观察（硬数据）

四个基线对"删除抵抗指令"的反应，平均 v2−v1 变化：

| 基线 | cell 数 | 平均 Δ | 模式 |
|---|---|---|---|
| 极简（海明威） | 5 | **+2.6** | 普遍暴涨 |
| 武侠（金庸） | 4 | +1.25 | 混合 |
| 反讽（王小波） | 5 | +0.4 | 几乎不动 |
| 清欢（汪曾祺） | 4 | +0.25 | 几乎不动 |

- **海明威基线**：4/5 cell 暴涨（+5/+4/+2/+2）。
- **反讽 + 清欢基线**：9 个 cell 里 8 个完全不变，仅各 1 个 +1~+2。
- **武侠基线**：武侠←反讽(6→6)、武侠←情欲(5→5) 在 v1 已接近天花板（无上升空间）；而 **武侠←网文 0/6 → 4/6（+4）** 是显著反转。

（注：反讽基线那个持续的"1/6"来自借词率维度。借词检测很敏感；句法 5 维全程守住。"反讽←极简"的 +2 来自句长/段长。海明威的极短句是强形式信号，把反讽产出也带短了一点，但幅度远小于海明威基线的崩塌。）

### 武侠←网文："假性弱传染"（源端的对应现象）

v1 矩阵里**网文是传染力最弱的源**（见 §2 发现 1，网文只造成过极少强信号）。但 `武侠←网文` 在 v1 下 0/6、v2 下 4/6，这意味着**去掉抵抗指令后，网文能打穿武侠了**。

说明 v1 测出的"网文弱传染"至少部分是**抵抗指令制造的假象**：当 prompt 说"你不会刻意模仿"，模型对网文这种"题材标记强、语言规则弱"的源会礼貌性地不学；指令一撤，它就大幅吸收。这是海明威"假性饱和"在**源端**的镜像——**基线端有"假性饱和"，源端有"假性弱传染"**，两者都是抵抗指令的产物。

（谨慎：这是单 cell 的强反转，方向明确但需要更多"弱源 × 各基线"的 v2 数据来确认普适性。）

### 真实现象记录（非推测）

1. **subagent 自我评价与文本统计分离**：海明威基线的 v2 产出，每篇 subagent 在任务总结里都自报"保持了冰山/省略风格"，但文本量化指标显示显著漂移（如 极简←纯爱 0/6 → 4/6）。**模型对自身风格的主观判断不能作为污染判据，只有落盘文本的统计可信。**

2. **模型从功能描述反推真实作者**：作家 background 是纯功能化描述，prompt 原文不含任何真实作者姓名（已用 grep 确认 writers/ 目录零命中"海明威"）。但 subagent 在英文总结里自发使用"Hemingway-style""Hemingway iceberg register"——模型从"冰山 / 省略 / 20 世纪中前期西方文学 / 狩猎男性气质"等特征反推出了海明威本人。

### 方法论 note

这个测试的结论经历了五次演化：基于 1 个 cell（极简←反讽 1→6）一度得出"海明威全靠抵抗指令"的颠覆性结论；补 3 个对照 cell 后修正为"只有海明威特殊"；补齐海明威基线全部 5 cell 后收敛到统计观察；补齐反讽基线对称的 5 cell 验证了机制方向；最后补齐清欢、武侠基线各 4 cell，得到四基线谱系并发现"假性弱传染"。**教训：单 cell 的漂移量不足以支撑普遍结论，对称的对照组才是关键。**

### 解释性假说（已获多基线数据支持；机制仍为推测）

**统计事实（硬数据，18 cell）**：四个基线对"删除同一句抵抗指令"的反应呈现谱系——极简 +2.6（暴涨）、武侠 +1.25（混合）、反讽 +0.4、清欢 +0.25（几乎不动）。海明威一端普遍被打穿，反讽/清欢一端纹丝不动。每点 n=20 可复现。

> ⚠️ 以下是对上述事实的**机制解释**，属推测——相关不等于因果，且仅单模型（Opus 4.7）、18 个 cell。

一个可能的解释：抗污染有**两种机制**——
- **提示词层面的"栅栏"**：海明威的风格是纯形式约束（句子 ≤ 15 字、不议论），模型清楚"如何关闭"它，v1 的抵抗指令是阻止切换的栅栏，去掉就垮（+2.6）。
- **风格层面的"内化"**：反讽（议论癖、哲学跳跃）和清欢（散文化、食物草木、平和收尾）是语义化的深层风格，不是开关式的，去不去抵抗指令都守得住（+0.4 / +0.25）。清欢与反讽同属内化型，互为佐证。

武侠是**混合型**（+1.25）：它对强源（反讽/情欲）在 v1 就被打穿到接近天花板，看不出栅栏效应；但对弱源（网文 0→4）暴露出栅栏成分。这引出源端的对应现象——**"假性弱传染"**：v1 测出的"网文弱传染"部分是抵抗指令的产物，撤掉后网文打穿了武侠。所以"假性"在基线端（海明威饱和）和源端（网文弱传染）同时存在，都是抵抗指令的造物。

要坐实因果，需跨模型复现 + 更多"纯形式约束型"作家（如只给句法规则、不给语义人格的合成作家）作对照，以及"弱源 × 多基线"的 v2 矩阵来验证"假性弱传染"的普适性。

## 5. 限制

1. **单模型**：只用 Claude Opus 4.7。换模型（Sonnet、GPT-4、Gemini）结论是否一致未测。
2. **虚构作家**：7 个风格基于真实文学传统的功能化抽取（避免模型直接调用记忆中的作者名）。代表性受限。
3. **借词率限制**：TF-IDF 指纹是基于阅读材料抽取的，对"借了源的句法但没借词"的污染不敏感。
4. **没做 LLM 裁判**：用户观察到的"摸了摸自己的脸"这类**叙事质感**漂移（动作类型、视角距离）超出了 5 个句法指标的覆盖范围。Phase 2 LLM 裁判可以做，但本实验范围内未做。
5. **抽象题目影响未控**：固定用"幻林折枝"做产出题目，没测过其他抽象题目（比如"无名"、"窗台"）下的漂移是否一致。
6. **功能化去名是"减速带"非"隔离墙"**：模型能从功能描述反推出真实作者（v2 实验中 subagent 自发说出"Hemingway"）。这不影响 v1/v2 的相对漂移测量（两边一致），但意味着"风格来源"无法完全归因于功能描述本身——模型可能一直在调用它认出的真实作者。

## 6. 后续工作

- **跨模型复现**：同样的 prompt 跑在 Sonnet 4.6、GPT、Gemini 上，看排行榜是否稳定。
- **LLM 裁判加入**：固定 schema 评分动作类型、视角距离、情绪温度、隐喻抽象度。
- **更多作家**：扩展到 12-15 位，包含更多"硬规则严密但词汇标志弱"的作家（比如卡夫卡式），检验饱和度假说的解释力。
- **题目鲁棒性**：换 3-5 个不同抽象度的产出题目，看 cell 排名是否保持稳定。
- **细粒度借词**：分词器版的借词率（用 jieba 切词后再算），看"概念词"是否比"字片段"更敏感。
